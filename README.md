# WMS仓库管理系统

## 技术栈
- 前端: Next.js 14 + React 18 + Tailwind CSS
- 后端: Supabase (PostgreSQL)
- 部署: Vercel

## 部署步骤

### 1. 创建Supabase
访问 supabase.com 创建项目

### 2. 执行数据库脚本
在SQL Editor中执行 supabase/migrations/001_initial_schema.sql

### 3. 部署到Vercel
1. 导入GitHub仓库
2. 配置环境变量
3. Deploy!
